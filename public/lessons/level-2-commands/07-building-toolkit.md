# Unit 2.07: Building Your Toolkit

**Level:** 2 - Slash Command Mastery
**Time:** 30 minutes
**Prerequisite:** Unit 2.06 - Command Optimization

---

## What You'll Learn

Organize 15-20 commands into a personal toolkit covering your daily work.

**Why This Matters:**
- Comprehensive coverage of daily tasks
- Organized by purpose
- Ready for any situation
- Maximum time savings

---

## Learn (5 minutes)

### Toolkit Categories

1. **Exploration** - Understand code
2. **Analysis** - Find issues
3. **Generation** - Create new code
4. **Fixing** - Solve problems
5. **Workflows** - Multi-step processes
6. **Maintenance** - Keep code healthy

### Organization Tips

**Use prefixes:**
- `explore-*` for exploration
- `analyze-*` for analysis
- `gen-*` for generators
- `fix-*` for fixes
- `check-*` for workflows

**Target:** 2-4 commands per category = 12-24 total
